Variable legend

W⁽ˡ⁾
Weight matrix for layer ℓ. Shape (n⁽ˡ⁾, n⁽ˡ⁻¹⁾). Each row = one neuron's weights.
b⁽ˡ⁾
Bias vector for layer ℓ. Shape (n⁽ˡ⁾, 1). One bias per neuron.

Pre-activation and activation. Shape (n⁽ˡ⁾, m). Columns = training examples.
dW⁽ˡ⁾, db⁽ˡ⁾, dZ⁽ˡ⁾
Gradients. Always same shape as their forward-pass counterpart.
X, Y, Ŷ
Input/label/prediction matrices. Shape (nₓ, m) and (n⁽L⁾, m).

The 3 golden rules

Rule 1 — W shape
W⁽ˡ⁾ = (n⁽ˡ⁾, n⁽ˡ⁻¹⁾)
Rows = current layer size, Cols = previous layer size. This makes W·A inner dimensions cancel.
Rule 2 — m columns everywhere
Z, A always have m columns. We process all m examples simultaneously (vectorization).
Rule 3 — gradients mirror forward shapes
dW has same shape as W, db same as b, dZ same as Z. The derivative of a scalar loss w.r.t. a matrix has the same shape as that matrix.
